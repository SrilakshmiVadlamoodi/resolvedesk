# Account Help

## Updating account details
Customers can update their name, email, and phone number from the "Account" section of
the VoltKart site. Email changes require confirming the new address via a verification
link before it takes effect. Phone number changes take effect immediately.

## Password reset
Password resets are self-service via the "Forgot password" link on the login page, which
sends a reset link to the account's registered email. The support agent cannot reset a
password directly or view a customer's password.

## Order history and saved addresses
Customers can view all past orders and manage saved shipping addresses from
"Account > Orders" and "Account > Addresses" respectively. Deleting a saved address does
not affect orders already placed with that address.

## Closing an account
Account closure requests are handled outside of chat for data-retention compliance
reasons — the support agent should escalate these requests rather than attempting to
process them directly.
