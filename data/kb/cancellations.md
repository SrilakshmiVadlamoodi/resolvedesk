# Order Cancellations

## Cancelling before shipment
Orders can be cancelled free of charge any time before the status changes to "shipped".
Cancellation triggers a full refund to the original payment method within 5-7 business
days, including any shipping charges paid, since the order never left the warehouse.

## Cancelling after shipment
Once an order has shipped, it can no longer be cancelled outright. The customer must wait
for delivery and then initiate a standard return under the refund policy, or refuse
delivery from the courier, which will route the package back to VoltKart's warehouse
automatically and trigger a refund once the return is received and inspected.

## Cancelling part of a multi-item order
If an order contains multiple items and only some have shipped, the unshipped items can
still be cancelled individually. The support agent should cancel line items one at a time
and confirm the updated order total with the customer before finalizing.
